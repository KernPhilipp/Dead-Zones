# Forest Facility Level Structure

- `blockout/`: early layout scenes and greybox geometry
- `environment/`: terrain, cliffs, roads, and exterior set dressing
- `buildings/`: research buildings, checkpoints, and interior shell scenes
- `props/`: reusable facility props and interactable set pieces
- `foliage/`: trees, bushes, ground cover, and forest dressing
- `spawn_points/`: player, zombie, and encounter marker scenes
- `lighting/`: outdoor lighting, fog, and atmosphere scenes/resources
- `navigation/`: navigation regions and pathing helpers
- `scripts/`: level-specific scripts
- `materials/`: level-only materials and shaders
- `audio/`: ambient forest, machinery, and facility sound assets
